# Changelog: F0001-npx-cli-installer

> **Date:** 2026-03-01 | **Branch:** feature/F0001-npx-cli-installer

## Summary

Implemented `fnd-cli` npm package distributed via `npx fnd-cli install`, enabling one-command FND installation into user projects. Includes interactive provider selection, automatic manifest generation, and update/uninstall subcommands with user-file preservation.
