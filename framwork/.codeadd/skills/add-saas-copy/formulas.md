# Copy Formulas

Frameworks to transform context into converting copy.

---

## PAS - Problem Agitate Solution

**Use:** Headlines, opening sections

### Structure

```
1. PROBLEM - Identify the pain
2. AGITATE - Amplify the consequence
3. SOLUTION - Present the way out
```

### Template

```markdown
[PAIN] is costing you [CONSEQUENCE].
[PRODUCT] solves this in [TIME/WAY].
```

### Examples

| Context | PAS Copy |
|---------|----------|
| Sales CRM | "Leads going cold in Excel? Every day without follow-up is lost money. EasyCRM organizes your pipeline in 5 minutes." |
| Project management | "Projects late again? Rework costs 3x more than preventing. TaskFlow shows bottlenecks before they blow up." |
| Automation | "Copying data between systems? Hours wasted every day. Zapify connects everything in 2 clicks." |

## BAB - Before After Bridge

**Use:** Storytelling, transformation sections

### Structure

```
1. BEFORE - Current situation (with pain)
2. AFTER - Desired situation (with benefit)
3. BRIDGE - How to get there (your product)
```

### Template

```markdown
Before: [CURRENT SITUATION WITH PAIN]
After: [DESIRED SITUATION]
Bridge: [PRODUCT] makes this transition.
```

### Examples

| Context | BAB Copy |
|---------|----------|
| Dashboard | "Before: manual reports every Monday. After: live data on mobile. DataViz is the bridge." |
| Onboarding | "Before: customers lost on first use. After: activation in 24h. OnboardKit is the bridge." |
| Support | "Before: tickets piling up. After: 80% resolved by bot. HelpDesk AI is the bridge." |

## 4Us - Headline Validation

**Use:** Test if headline is strong enough

### Criteria

| U | Question | If NOT |
|---|----------|--------|
| **Urgent** | Creates sense of urgency? | Add consequence of not acting |
| **Unique** | Different from what others say? | Specify your differential |
| **Ultra-specific** | Has concrete numbers/details? | Add stats, time, % |
| **Useful** | Shows clear benefit? | Focus on result, not feature |

### Application

```markdown
## Original Headline
"Manage your projects easily"

## 4Us Test
- Urgent? NO - no consequence
- Unique? NO - any tool says this
- Ultra-specific? NO - "easily" is vague
- Useful? WEAK - "manage" is not a benefit

## Improved Headline
"Stop missing deadlines. Setup in 5 min, no training."

## 4Us Test
- Urgent? YES - "missing deadlines" is a consequence
- Unique? YES - "no training" is a differential
- Ultra-specific? YES - "5 min"
- Useful? YES - clear benefit (not missing deadlines)
```

## F-B-O - Feature Benefit Outcome

**Use:** Transform technical features into sales arguments

### Structure

```
FEATURE: What the product DOES (technical)
BENEFIT: What this MEANS for the user
OUTCOME: The RESULT in the user's life/work
```

### Template

```markdown
[FEATURE] so you can [BENEFIT], resulting in [OUTCOME].
```

### Conversion Table

| Feature | Benefit | Outcome |
|---------|---------|---------|
| Real-time dashboard | See metrics live | Make faster decisions |
| No-code automations | Create flows without programming | Save hours per week |
| Slack integration | Receive alerts where you work | Never miss important notifications |
| Automatic reports | No manual report writing | Have Mondays free |
| Multi-workspace | Separate projects by client | Stay organized even while growing |
| Change history | Know who changed what | Resolve conflicts in seconds |

### Full Example

```markdown
## Feature: RESTful API with webhooks

**Wrong (feature only):**
"Complete RESTful API with webhook support"

**Better (feature + benefit):**
"Integrate with any tool you already use"

**Ideal (feature + benefit + outcome):**
"Connect your entire stack in an afternoon. Your data flows automatically between systems — no more copy-pasting ever again."
```

## When to Use Each Formula

| Situation | Formula | Why |
|-----------|---------|-----|
| Main headline | PAS | Captures attention through pain |
| "How it works" section | BAB | Shows transformation |
| Validate headline | 4Us | Ensures it's strong |
| List features | F→B→O | Transforms technical into benefit |
| Testimonials | BAB | Customer tells transformation |
| CTA section | PAS | Urgency to act |

## Combining Formulas

**Headline (PAS):**
> "Leads going cold in Excel? EasyCRM organizes your pipeline in 5 min."

**Subtitle (F→B→O):**
> "Visual pipeline + automations so you close more, typing less."

**CTA section (BAB):**
> "Leave the spreadsheet chaos behind. Get full control of your pipeline. Start free today."
